Web Application Development

Course Code: ITM 1131

Course Level: Year One, Semester One

Credit Unit: 4

1. Course Discription.

This course introduces students to Web technologies - HTML, CSS, JavaScript; CSS frameworks;
JavaScript frameworks. The students will learn to implement modern web pages using HTML and CSS
framework. The course also introduces students to full stack web application programming using
JavaScript.

2. Course Objectives.

Upon successfully completing this course, students should be able to:

  • describe web technologies (HTML, CSS & JavaScript), CSS frameworks and JavaScript frameworks.
  • implement a simple but modern web page with HTML and CSS.
  • build a simple but functional web application using JavaScript.

3. Learning Outcomes.

Upon successfully completing this course, students can:

  • Use web technologies (HTML, CSS & JavaScript), CSS frameworks and JavaScript frameworks
  • Design and create a simple but functional web page using HTML and CSS framework.
  • Compose and build a simple full stack web application using JavaScript framework.

5. Mode of Delivery

The mode of delivery will include, but are not limited to lecture sessions, laboratory sessions, tutorial
sessions, project work. The deliveries will take face-to-face, online, and self-study mode.

6. Methods of Assessment

Mode of assessment will include (i) formative assessments: - tests, assignments, homework; (ii)
summative assessments: - mid-semester examination, end of semester examination, e.g, written, viva,
project report, practical; (iii) 'Authentic' or work-integrated assessment: - projects, laboratory practicals.
Relative contributions of the assessment to final grade are:

  • Formative assessments (Assignments, Homeworks): 10%
  • Work-integrated assessment (Projects, Laboratory Practical): 20%
  • Summative assessment (2 Semester Tests in Week 6 & 12): 20%
  • Summative assessment (End of Semester Examination): 50%
